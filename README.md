# Agente_Hardware_Fase_1
Proyecto donde se diseño un robot de movimiento autónomo funcional en algún punto de una ciclovía dentro de CU BUAP.  Este repositorio solo tiene los códigos necesarios  para los ESP32, ESP32 CAM y un PC que funciona como cerebro (Python). 

En el caso de los archivos .ino es necesario tener las librerías utilizadas, y contar con una versión del esp32 menor a la 2.15. Para el archivo python requiere descargar las bibliotecas y el modelo Yolo (ese se instala solo al tener ultralytics). A la hora de hacer la conexión todos los elementos deben estar conectados a la misma red WiFi.
